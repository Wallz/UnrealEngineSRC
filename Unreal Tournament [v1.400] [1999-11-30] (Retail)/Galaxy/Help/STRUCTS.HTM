glxBuffer

typedef struct
{
udword FourCC;
udword Size;
void *Handle;
ubyte *Data;
udword Length;
udword PlayPos;
udword WritePos;
sdword Premix;
} glxBuffer;

Fields

FourCC

FourCC value used for identification of structure, should be GLX_FOURCC_BUFF.

Size

Size in bytes of the rest of the structure, that is starting AFTER the Size field.

Handle

Pointer to Directsound secondary buffer object (LPDIRECTSOUNDBUFFER) or pointer to WaveOut header (LPWAVEHDR), check glxOutput::Driver field to see handle used.

Data

Pointer to actual audio data.

Length

Length of buffer in bytes.

PlayPos

Current play position(=sample heard) in bytes.

WritePos

Current write position(=first sample which is not queued and thus can be modified) in bytes.

Premix

Bytes between current play position and current mixed MUSIC samples, this effectively gives the delay (in bytes) before a new note is heard from the speakers.

glxOutput

typedef struct
{
udword FourCC;
udword Size;
void *Handle;
void *Mixer;
void *Listener;
void *Extensions;
ubyte Driver;
ubyte Format;
ubyte Type;
ubyte Reserved;
} glxOutput

Fields

FourCC

FourCC value used for identification of structure, should be GLX_FOURCC_OUTP.

Size

Size in bytes of the rest of the structure, that is starting AFTER the Size field.

Handle

Pointer to Directsound object (LPDIRECTSOUND) or handle to WaveOut device (HWAVEOUT), check Driver field to see handle used.

Mixer

Pointer to Directsound primary buffer object (LPDIRECTSOUNDBUFFER) or NULL.

Listener

Pointer to Directsound 3D listener object (LPDIRECTSOUND3DLISTENER) or pointer to Galaxy G3D listener (glxVector *), check Type field to see listener used.

Extensions

Pointer to A3D extension object (LPIA3D), pointer to EAX propertyset (LPKSPROPERTYSET) or NULL, check Type field to see extension used.

Driver

Output driver used, can be GLX_DIRECTSOUND or GLX_WAVEDRIVER.

Format

Output format used by driver, can be any combination of GLX_STEREO, GLX_MONO, GLX_8BIT or GLX_16BIT.

Type

Output driver type, can be GLX_NORMAL, GLX_G3D, GLX_A3D or GLX_EAX.
