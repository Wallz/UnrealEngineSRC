glxSetCDAudioVolume

The glxSetAudioCDVolume function sets the overall CD Audio volume (this does NOT affect the music and/or sample volume).

int glxSetCDAudioVolume
(
int Volume,
int Speed
)

Parameters

Volume

Specifies the overall CD Audio volume (0..127).

Speed

Specifies the CD Audio volume change speed in units per millisecond. The following values are defined :

GLX_VOLSET

The overall CD Audio volume is changed immediately.

GLX_VOLFADE

The overall CD Audio volume is changed by one (1) unit per millisecond (giving a full scale ramp time of 128 milliseconds)

GLX_VOLFADE2

The overall CD Audio volume is changed by two (2) units per millisecond (giving a full scale ramp time of 64 milliseconds)

Return values

Returns GLXERR_NOERROR if successful or one of the following errors otherwise :

GLXERR_BADPARAMETER
